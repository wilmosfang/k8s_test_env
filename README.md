## 基础软件依赖

需要系统中已经安装如下两个软件

* **virtualbox 6.0.10**
* **vagrant 2.2.5**


## 安装方法

~~~
git clone https://github.com/wilmosfang/k8s_test_env.git
cd k8s_test_env
vagrant up
~~~
